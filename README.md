# Sudarshan
Real-time hospital emergency management system built with Node.js and MongoDB.
A web-based emergency management system that helps users quickly locate nearby hospitals, check availability, and manage emergency response efficiently.

Sanjeevini is designed to:

Help users find nearby hospitals during emergencies
Display hospital details (beds, ICU, contact info, etc.)
Provide real-time emergency information
Assist hospitals in managing emergency capacity

This system aims to reduce response time and improve coordination during medical emergencies.

Problem Statement:
During emergencies, people often struggle to:
Find nearby hospitals quickly
Know bed/ICU availability
Contact hospitals efficiently
Get real-time emergency updates

This project provides a centralized platform to solve these issues.

Tech Stack
Frontend:
HTML5
CSS3
JavaScript (Vanilla)

Backend:
Node.js / Express.js

Database:
MongoDB

APIs:
Google Maps API (for location services)

Features:

🔍 Search nearby hospitals

📍 Location-based results (Google Maps / Geolocation)

🏥 View hospital details (beds, ICU, emergency services)

📞 Quick contact access
